Преобразовывает дату в строку.

public static convertDateToString(array<string|int, mixed> &$array, array<string|int, string|int> $keys) : void

Преобразовывает сущность объекта Bitrix\Main\Type\DateTime в строку.

Находит в исходном массиве поля, соответствующие ключам, которые были переданы в качестве аргумента функции, и формирует для этих полей строку, содержащую дату.

Если не удалось преобразовать дату к строке, полю присвается пустая строка.

Преобразования осуществляются в массиве, который передан в качестве аргумента функции.

Parameters
$array : array<string|int, mixed>

Массив, который необходимо преобразовать.

$keys : array<string|int, string|int>

Ключи элементов массива $array, в которых хранятся даты.

convertElementUrl()

IblockQueries.php : 94

Формирует URL-адрес по шаблону URL-адреса.

public static convertElementUrl(array<string|int, mixed> &$element_array[, string $detail_url = '' ][, string $section_url = '' ][, string $iblock_url = '' ]) : void

Преобразовывает шаблоны URL-адреса страницы детального просмотра, URL-адреса страницы раздела и URL-адреса страницы инфоблока.

Адреса хранятся в полях массива с ключами "DETAIL_PAGE_URL", "SECTION_PAGE_URL" и "LIST_PAGE_URL" соотвественно. В исходном массиве должны присутствовать эти поля. Также в исходном массиве должны присутствовать поля, которые используются в шаблонах адресов (ID, SECTION_ID, IBLOCK_ID и др.).

Если в функцию переданы аргументы с шаблонами адресов, будут использованы эти шаблоны. В противном случае будут использованы шаблоны, заданные в параметрах инфоблока.

Если не удалось преобразовать шаблон URL-адреса, полю присвается пустая строка. Преобразования осуществляются в массиве, который передан в качестве аргумента функции.

Parameters
$element_array : array<string|int, mixed>

Массив полей элемента.

$detail_url : string = ''

Шаблон адреса страницы детального просмотра. Необязательный параметр.

$section_url : string = ''

Шаблон адреса страницы раздела. Необязательный параметр.

$iblock_url : string = ''

Шаблон адреса страницы инфоблока. Необязательный параметр.
